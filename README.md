## Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/franciswagonda/franciswagonda.github.io.git
   cd franciswagonda.github.io
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server (recommended):
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js
     npx serve
     ```

3. **View the site**
   - Navigate to `http://localhost:8000` (or the port shown)
